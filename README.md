# Build/Run Instructions

## Build Instructions
1. Make a file called .env
2. Fill .env with the information below
   DB_HOST="localhost"
   DB_NAME="csc317db"
   DB_USER='root'
   DB_PASSWORD='kcsports1000@'
   PORT=3000
3. Now do 
   npm i 
   to install
4. then do
   npm run builddb
   to setup database

## Run Instructions
1. cd application
2. npm start
3. On your browser, type in localhost:3000
4. Register for an account 
5. Login to your account
6. Get started uploading your videos
